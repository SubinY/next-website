# Shadcn Website Build with the Next.js 14

![首页](./public/readme.png)

## Overview

通过以下库快速搭建nextjs个人网站

- Framework: Next.js 14
- Styling: Tailwind CSS
- Headless UI: shadcn-ui、radix-ui、magic-ui、aceternity-ui
- Database: Postgres
- ORM: sequelize
